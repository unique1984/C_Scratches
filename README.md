**08.08.2018** hex2rgb.c

* **Building**
	* gcc hex2rgb.c -o hex2rgb
* **Using**
	* ./hex2rgb FF00FF

```
yasin@uxn:~/C_Education/C_Scratches$ ./hex2rgb FF00FF
Hex24   #FF00FF
RGB888 (255, 0, 255)

Hex16   #F81F
RGB565 (31, 0, 31)
```
